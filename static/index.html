Règles du jeu

nombre de dés : 8

nombre de tentatives: 3

Jeter les dés un par un. On peut s'arrêter à tout moment. La tentative se termine après la lancée des 8 dés.

Obtenir un "un" entraine que la tentative est invalide

Score: somme de la valeur de tous les dés lancés par le joueur dans une tentative.

Lancez les dés un à un. À tout instant, vous pouvez décider d’arrêter et de conclure là votre tentative. Autrement, celle-ci s’achève lorsque vous avez lancé les huit dés.
Attention : si vous obtenez un 1, vous échouez dans votre tentative.
Le résultat est la somme des valeurs de tous les dés lancés.
Vous avez 3 tentatives pour obtenir le meilleur résultat possible.
